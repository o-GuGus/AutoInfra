# AutoInfra

- **Simply to install**
> For Debian like

~~~shell
cd /tmp
wget https://raw.githubusercontent.com/o-GuGus/AutoInfra/master/deploy.sh
chmod +x deploy.sh
./deploy.sh
~~~

- Name Server 1
- Name Server 2
- AD DC Primary
